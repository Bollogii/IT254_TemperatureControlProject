# IT254_TemperatureControlProject
Final Project for the college class IT254, hardware and software. Project was to integrate AI or machine learning into an Arduino based project. My group chose to make an AI powered temperature and humidity controller. As a proof of concept we used a simple fan and colored LEDs to show the code reacting to input from a sensor. 
The images are what we wired for the project and the code can be found as an .ino file.
Code for our arduino is in C++
we used an NPN transistor as control for the fan due to its need for a direct 5v source.
